My TranSIT

My TranSIT is a Full Stack Application that was created with the active commuter in mind. We wanted to create an application that will give you the fastest route to your destination, and will alert you ahead of time if you will arrive late. This application uses the Google Maps API to take in the user's start and end points, and display back to them the fastest route. I played a large role working on the back end for this project by working heavily with the API functionality, and having the best routes and their directions properly to the page. Created by Nicole Apsche, Dominique Griffith and Danny Montano.

Technologies Used

HTML, CSS, JavaScript, jQuery, Google Maps API, Firebase, Bootstrap 4
