Git-A-Pet

Git-A-Pet was created to be a user-friendly platform for people to search for adoptable animals. Espeically with the pandemic, many people are now working remotely permanently, and are looking for some furry coworkers! The application allows you to search animals based on animal type, location, age, and gender. Results are then displayed to the user, along with links to the external adoption sites for more information. We built this entire application together, and worked really well as a team. Created by Nicole Apsche, Margaret Egner, and Solomon Lee.

Technologies Used

HTML, CSS, JavaScript, jQuery, Bootstrap 4, PetFinder API, MySQL, Sequelize
