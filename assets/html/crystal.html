Crystal Collector

Crystal Collectors was created to demonstrate a simple game that could be made using JavaScript and jQuery. When the page loads, the game generates a random number to be your goal. Each of the crystals holds a hidden value that will be added to your score. The object of the game is to match your score to the random value. Careful, go over the random number and you lose! When a new game begins, all the numbers are randomized.

Technologies Used

HTML, CSS, JavaScript, jQuery, Bootstrap 4
